# PDF Crashes After Update - How to Roll Back and Fix

**PDF reader crashing after recent update?** This guide shows how to roll back updates, fix compatibility issues, and restore stable PDF viewing.

## Why Updates Cause Crashes

### Breaking Changes in New Versions
Adobe and other PDF readers occasionally introduce bugs in updates that conflict with specific OS versions or hardware configurations.

### Incompatible Plugins
Third-party plugins compatible with older versions may break after updates, causing instant crashes.

### Corrupted Update Installation
Interrupted updates leave partial installations with missing DLLs or broken registry entries.

## Rollback Solutions

**Windows - Uninstall Recent Update**:
```
Control Panel → Programs → Uninstall an update → Select Adobe Reader update → Uninstall
```

**Mac - Time Machine Restore**:
Restore /Applications/Adobe Acrobat Reader DC from pre-update backup.

**Download Previous Version**:
Visit filehorse.com or oldversion.com for older stable releases.

## Fix Without Rollback

**Repair Installation**: Windows → Programs and Features → Adobe Reader → Change → Repair
**Clear Cache**: Delete %AppData%\Adobe and %LocalAppData%\Adobe folders
**Disable Auto-Update**: Preferences → Updater → "Do not download or install updates automatically"
