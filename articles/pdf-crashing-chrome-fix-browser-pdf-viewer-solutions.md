# PDF Crashing Chrome Fix - Browser PDF Viewer Solutions

**PDFs crashing Chrome browser?** Fix Chrome's built-in PDF viewer and browser integration crashes.

## Chrome PDF Fixes
1. Disable built-in viewer: `chrome://settings/content/pdfDocuments`
2. Clear browser cache
3. Disable extensions
4. Reset Chrome settings

## Alternative Solutions
- Use external PDF reader
- Try Microsoft Edge
- Download PDFs instead of opening
