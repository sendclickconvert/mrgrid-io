# PDF Won't Load Just Crashes - Loading Failure Diagnostics

**PDF starts loading then crashes?** Diagnose and fix load-time rendering failures.

## Diagnostics
```bash
qpdf --check suspicious.pdf
```

## Fixes
1. Disable Protected View
2. Update graphics drivers
3. Linearize PDF: `qpdf --linearize broken.pdf fixed.pdf`
4. Try alternative readers
