# Fix PDF Crashing Issue - Universal Solutions for All Platforms

**PDF crashes on any device?** This comprehensive guide provides platform-agnostic solutions that work across Windows, Mac, Linux, Android, and iOS.

## Universal Diagnostic Steps

### 1. Isolate the Problem Source
- **Test multiple PDFs**: If only one crashes, the file is corrupted
- **Test multiple readers**: If all crash, system-level issue
- **Test different devices**: If crashes everywhere, file corruption

### 2. Quick Universal Fixes

**Update Everything**:
- PDF reader to latest version
- Operating system to latest patch
- Graphics drivers (desktop platforms)

**Clear Application Data**:
- Windows: %AppData% and %LocalAppData%
- Mac: ~/Library/Preferences and ~/Library/Caches
- Android: Settings → Apps → PDF Reader → Clear Cache
- iOS: Uninstall and reinstall app

**Check Available Storage**:
Minimum 2GB free space required for PDF temp files.

### 3. Platform-Specific Solutions

**Windows**: Disable Protected Mode, update .NET Framework
**Mac**: Reset PRAM/NVRAM, repair disk permissions
**Android**: Disable battery optimization for PDF app
**iOS**: Reset All Settings (preserves data)

## Advanced Repair Methods

**Corrupt PDF Recovery**: Use QPDF, PDFtk, or online repair tools
**Alternative Readers**: SumatraPDF, Foxit, PDF-XChange
**System Repair**: sfc /scannow (Windows), Disk Utility (Mac)
