# PDF File Crash on iPhone - iOS PDF Viewer Fix Guide

**PDFs crashing on iPhone?** Fix iOS Files app and third-party PDF reader crashes.

## iOS PDF Fixes
1. Restart iPhone
2. Update iOS
3. Clear Files app cache
4. Reinstall PDF reader app

## App-Specific Solutions
**Files App**: Settings → General → iPhone Storage → Files → Offload App
**Adobe Acrobat**: Delete and reinstall from App Store
**Alternative**: Use Apple Books for PDFs
